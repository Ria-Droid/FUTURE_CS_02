# Phishing Detection & Awareness Report

## 👤 Author
Ria Khan

## 📌 Overview
This project focuses on analyzing phishing email samples and identifying common attack techniques used by cybercriminals.

## 🎯 Objectives
- Identify phishing indicators
- Classify email risks (Safe / Suspicious / Phishing)
- Explain phishing attacks in simple terms
- Provide prevention and awareness guidelines

## 🛠️ Tools Used
- Public phishing email samples
- Email header analysis tools (reference)
- Manual inspection of links and domains

## 🔍 Analysis Approach
1. Collected phishing and legitimate email samples
2. Analyzed email structure, tone, and links
3. Identified phishing indicators
4. Classified emails into Safe, Suspicious, and Phishing
5. Developed user awareness guidelines
